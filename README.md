# PitchSense Test

AI-powered roleplay evaluation and analysis platform.

**Live Demo:** [pitchsense-assessment.vercel.app](https://pitchsense-assessment.vercel.app)

## Features

- Evaluate roleplay sessions with AI
- Detailed performance metrics (Communication, Empathy, Product Knowledge)
- Risk assessment and recommendations
- Real-time analysis powered by Groq AI

## Setup

1. Install dependencies: `npm install`
2. Set `GROQ_API_KEY` in `.env`
3. Run: `npm run dev`

Built with Next.js, TypeScript, and Tailwind CSS.
